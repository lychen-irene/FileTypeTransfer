# Converting All Files in A Specific Directory by Powershell

## CSV to XLSX

`$ powershell -ExecutionPolicy Bypass  {DocumentPath}\csv2xlsx.ps1`

## XLS to XLSX

`$ powershell -ExecutionPolicy Bypass {DocumentPath}\xls2xlsx.ps1`

## Convert and Delete Original Files at the Same Time

`$ .\csv2xlsx.ps1 -DeleteOriginal`
`$ .\xls2xlsx.ps1 -DeleteOriginal`

## Convert all XlSX File into PDF File by Giving Source Path

`$ powershell -ExecutionPolicy Bypass xlsx2pdf.ps1 -InputPath "{XLSX File SourcePath}" -Batch`

- suggest adding `-ExecutionPolicy Bypass` for permission issue
